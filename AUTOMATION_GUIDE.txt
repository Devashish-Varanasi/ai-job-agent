================================================================================
AI JOB AGENT - AUTOMATION GUIDE
================================================================================

SINGLE COMMAND AUTOMATION - Just run one file and everything is done!

================================================================================
MAIN COMMAND
================================================================================

Run this single command:
    python run_automation.py

What it does automatically:
  1. Parses your resume and extracts your name
  2. Auto-detects the job role from your resume skills
  3. Fetches matching jobs from Adzuna API
  4. Generates personalized cover letters for ALL jobs
  5. Creates UNIQUE filenames every time you run it

================================================================================
OUTPUT FILES - UNIQUE EVERY TIME
================================================================================

CSV File (in outputs/ folder):
  Format: jobs_<JobType>_<Timestamp>.csv
  Example: outputs/jobs_data_analyst_20251110_143025.csv
  
Cover Letters Folder (in cover_letters/ folder):
  Format: <YourName>_<TodayDate>/
  Example: cover_letters/John_Doe_2025-11-10/
  
  Inside this folder:
    - CoverLetter_<Company>_<JobTitle>_<JobID>.docx (for each job)
    - Example: CoverLetter_Google_Data_Analyst_123456.docx

================================================================================
OPTIONAL PARAMETERS
================================================================================

Manual job type:
    python run_automation.py "software engineer"

With location:
    python run_automation.py "data scientist" "Bangalore"

================================================================================
WHAT HAPPENS EACH RUN
================================================================================

Run 1 (10:30 AM):
  - outputs/jobs_data_analyst_20251110_103045.csv
  - cover_letters/John_Doe_2025-11-10/
    (Contains all cover letters as DOCX files)

Run 2 (2:45 PM - same day):
  - outputs/jobs_data_analyst_20251110_144532.csv
  - cover_letters/John_Doe_2025-11-10/
    (Adds to the same folder if same day, or creates new if different day)

Run 3 (Next day):
  - outputs/jobs_data_analyst_20251111_090015.csv
  - cover_letters/John_Doe_2025-11-11/
    (New folder for new date)
